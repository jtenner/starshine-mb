# Research Archive

This directory holds repo-authored numbered investigations that are no longer kept in `docs/`.

- Keep the original numbered filename stable when moving a research note here.
- Move a research note here once its durable conclusions live in `docs/wiki/` or it otherwise stops being the active normative contract.
- Repoint live references when moving a file here so backlog notes, changelog entries, and wiki sources stay valid.
- Treat archived research notes as source material, not the main living answer.
