# ADR Format

ADRs live in `docs/adr/` and use sequential numbering like `0001-slug.md`.

## Minimal template
```md
# {Short title}

{1-3 sentences: context, decision, and why.}
```

Optional additions only when useful:
- status frontmatter
- considered options
- consequences

## Offer an ADR only when all are true
1. hard to reverse
2. surprising without context
3. result of a real trade-off
