# Scaffold Exercises

Create exercises from existing material.

## Workflow
1. Identify the concept to teach.
2. Define the starting state the learner should receive.
3. Write the task prompt and constraints.
4. Create checkpoints or acceptance criteria.
5. Separate starter material from solution material.
6. Keep each exercise focused on a single learning objective.
