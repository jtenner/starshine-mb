# To Issues

Convert source material into actionable issues.

## Workflow
1. Read the source material completely.
2. Identify distinct deliverables.
3. Split work into issues that can be completed and reviewed independently.
4. For each issue, include summary, motivation, acceptance criteria, and explicit non-goals.
5. Prefer fewer, clearer issues over many tiny ones.
