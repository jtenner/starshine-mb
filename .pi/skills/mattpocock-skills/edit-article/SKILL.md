---
name: edit-article
description: Edit and improve articles by restructuring sections, improving clarity, and tightening prose. Use when user wants to edit, revise, or improve an article draft.
---

1. Split the article into sections using headings.
2. Identify the key point of each section and the dependency order between ideas.
3. Reorder sections if needed so later ideas depend only on earlier ones.
4. Confirm the structure with the user.
5. Rewrite each section for clarity, coherence, and flow.
6. Keep paragraphs tight; prefer short paragraphs and direct sentences.
