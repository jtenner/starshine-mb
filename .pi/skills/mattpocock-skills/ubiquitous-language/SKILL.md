# Ubiquitous Language

Develop and refine the project's shared language.

## Workflow
1. collect important domain terms from code, docs, and discussion
2. identify synonyms, collisions, and ambiguous words
3. propose canonical terms with short definitions
4. record avoid-terms and rationale
5. update docs when terminology is resolved
