# Obsidian Vault

When working with an Obsidian vault:
1. Ask what the user wants to know or update.
2. Search the vault for relevant notes, aliases, tags, and backlinks.
3. Summarize the most relevant context before acting.
4. Create or update notes when durable memory should be captured.
5. Prefer extending existing notes over creating near-duplicates.
