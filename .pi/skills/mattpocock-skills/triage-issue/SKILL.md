# Triage Issue

For a single issue:
1. read the full issue and comments
2. summarize the problem in plain language
3. clarify expected vs actual behavior
4. try to reproduce when possible
5. identify missing info, likely cause, and scope
6. recommend the next state and any follow-up questions
